# Mailbox UI Assignment

A responsive Mailbox UI application built using React, React Router, Tailwind CSS, and React Icons.

## Features

- Responsive sidebar
- Inbox, Sent, Draft, and Trash pages
- Search functionality
- Mail details view
- Attachments support
- Responsive mobile design
- Modern UI design

## Technologies Used

- React.js
- React Router DOM
- Tailwind CSS
- React Icons

## Installation

bash
git clone https://github.com/D-Nagalakshmi/mailbox-ui-assignment.git
cd my-react-app
npm install
npm run dev
